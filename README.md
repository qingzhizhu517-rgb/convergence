# Convergence (收敛式卡片笔记)

`Convergence` 是一款专为 Android 平台打造的**本地双向链接卡片笔记与 Markdown 编辑器**。结合了 Material Design 3 现代美工设计，旨在为您提供最纯粹、安全且极速的知识沉淀体验。

---

## ✨ 核心特性

### 1. 🔗 强大的双向链接 (Bidirectional Hyperlinks)
* **`[[WikiLinks]]` 语法支持**：通过简单的双括号链接，将零散的知识点无缝织网。
* **反向跳转与自愈**：在阅读笔记时，点击内链可直接精准跳转。如果引用的笔记尚不存在，系统会自动提示并引导您一键生成，杜绝断链。

### 2. 📂 随心的文件管理与“零碎笔记”机制
* **多层级文件夹分类**：支持创建、重命名、删除个性化文件夹。
* **完美的“零碎笔记”（Loose Notes）收容所**：不属于任何文件夹的笔记将自动归档为零碎笔记，随写随记，轻装上阵。
* **随时随地的迁移**：在阅读笔记时，通过顶部的快捷操作，即可瞬间将当前笔记移入另一文件夹或移回零碎笔记状态。

### 3. ✍️ 沉浸式 Markdown 渲染与编辑
* **多功能排版支持**：完美呈现标题（H1-H6）、粗体、斜体、代码块、列表甚至 Markdown 复杂的表格渲染。
* **文档大纲导航 (Table of Contents)**：为长篇笔记自动提取生成层级目录面板（TOC），支持点击目录快速描点跳转。
* **双模流畅切换**：编辑与预览一键触屏即达。

### 4. 🚀 外部文件极速导入
* **一键导入外部 Markdown**：支持将手机本地的 `.md` / `.txt` 文件直接加载进数据库。
* **标题自动解算**：系统会智能分析文件头部内容，自动提取 `#` 标题作为文件名。如果重名，会自动开启带有增量序号的安全重命名机制（如 `Book (1)`），防止旧笔记被意外覆盖。

### 5. 🔒 100% 本地安全
* **完全离线运行**：基于 Android 的 Jetpack Room 本地数据库，您的数据仅属于您自己。
* **极速数据库联动**：对所有文件删除（采用强安全级级联空置 `SET_NULL`）和数据库更新加入了高级异常过滤保护，全面消除异常闪退。

### 6. 🌐 智能双语与优雅设计
* **全面支持中英文**：在设置面板一键切换，全局界面秒级自适应。
* **Material Design 3 标准**：采用考究的字体间距、呼吸感十足的 negative space（留白）以及极佳的暗蓝/亮白对比色彩搭配，打造艺术品级的阅读享受。

---

## 🛠️ 技术架构

* **编程语言**：100% Kotlin (Modern Coroutines & StateFlow)
* **界面框架**：Jetpack Compose (Edge-to-Edge 全屏触控，响应式自适应布局)
* **数据存储**：SQLite via Room Database (带安全事务容错级)
* **依赖注入**：轻量级依赖注入与 MVVM (Model-View-ViewModel) 流式单向数据架构

---

## 📥 开始使用

1. 打开应用，您可以在左侧或底部导航栏在 **我的文件夹**、**全局搜索** 与 **设置** 间丝滑切换。
2. 每一个打开的笔记均支持双击内链跳跃。在阅读长文时，随时点击右上角折叠大纲进行快速翻阅。
3. 不需要复杂繁琐的注册。Convergence 没有云服务器，绝不会上传隐私，保障您极简纯粹的数字墨水体验。
